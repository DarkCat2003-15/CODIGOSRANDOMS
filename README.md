# CODIGOSRANDOMS

Repositorio donde almaceno diferentes ejercicios, pruebas y pequeños proyectos realizados durante mi proceso de aprendizaje y práctica en desarrollo de software en una empresa.

## Contenido

Actualmente este repositorio incluye:

- 📁 **DOC**  
  Documentación y archivos de apoyo.

- 📁 **PRUEBA HTML Y JAVA**  
  Ejercicios básicos utilizando HTML, CSS y JavaScript.

- 📁 **Prueba1**  
  Prácticas realizadas en Java.

- 📁 **Prueba2**  
  Ejercicios de programación orientada a objetos en Java.

- 📁 **Prueba3 BD**  
  Scripts y ejercicios de bases de datos SQL.

- 📄 **Angular-Basico.postman_collection.json**  
  Colección de Postman para pruebas de APIs.

- 📦 **angular-0.0.1-SNAPSHOT.jar**  
  Archivo ejecutable generado durante pruebas de un proyecto Spring Boot.

## Objetivo

Este repositorio funciona como un espacio para:

- Practicar nuevos conceptos.
- Experimentar con diferentes tecnologías.
- Mantener un registro de mi progreso.
- Guardar ejercicios y pruebas de desarrollo.
